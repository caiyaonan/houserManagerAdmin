# 房屋管理后台

简体中文 | [English](./README.md)

> 毕设房屋管理后台

## 介绍

**基于Vue-Admin开发的房屋管理后台**

### 功能

1. 管理员管理
2. 用户管理
3. 房屋管理
4. 消息管理

## 安装和运行

```bash
# 克隆库
git clone https://github.com/caiyaonan/houserManagerAdmin.git

# 进入目录
cd houserManagerAdmin

# 安装依赖
npm install

# 运行
npm run dev
```
## 编译

```bash
# 使用测试环境的配置编译
npm run build:stage

# 使用生产环境的配置编译
npm run build:prod
```
